Ehbanana (pronounced "a banana") is a web based GUI framework built with C++ and javascript without any large libraries such as node modules. It also uses the system's browser instead of packaging it inside.

Input and Output

The various HTML5 inputs generate a message that is sent to the backend for processing. The backend sends back a message with a response to output
